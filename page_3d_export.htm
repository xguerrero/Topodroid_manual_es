VISOR 3D - Exportar

El diálogo se abre desde el menú Exportar de la ventana 3D.

El modelo 3D se puede exportar como

  • glTF (formato de transferencia GL)
  • CGAL
  • STL, binario o ascii
  • LAS, binario
  • DXF
  • KML (si el modelo está georreferenciado)
  • Shapefile (si el modelo está georreferenciado)

Cuando sea compatible con el formato, el archivo exportado puede incluir

  • Estaciones
  • Visuales
  • Paredes (si se generan)
  • de superficie (si está cargado)
