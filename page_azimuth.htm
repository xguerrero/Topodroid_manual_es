AMPLIAR REFERENCIA

Este cuadro de diálogo se abre desde el botón circularBrújula en la ventana de datos de la topografía o en la ventana de croquis.

La referencia de declinación se usa para asignar la dirección de "declinación" del perfil a las tomas de la línea media.
La dirección de "declinación" no se establece automáticamente para las extensiones.

El botón brújula muestra una flecha larga dentro de un círculo, orientada hacia el dirección de referencia de declinación actual, o una flecha corta dirigida hacia la izquierda o hacia la derecha (para "declinación" fija).
Cuando el perfil "declinar" se calcula utilizando la "referencia de declinación", se establece a la derecha si difiere del referencia por menos de 90°. De lo contrario, se establece a la izquierda.

Cuando la configuración Referencia de declinación fija está habilitada, el botón brújula tiene solo dos estados, IZQUIERDO o DERECHO. Al tocar el botón se alterna entre ellos. La nueva declinación de tomas se establece según el estado. del botón.

Este diálogo tiene

  • "ampliar referencia" brújula
  • control deslizante para cambiar su valor entre -180 y 180°
  • campo de texto para introducir el valor numérico del brújula, entre 0 y 360
  • brújula establece el valor utilizando los sensores magnéticos de Android
  • izquierda obliga a "declinar" a IZQUIERDA (-1)
  • derecha obliga a "declinar" a la DERECHA (+1)
  • ok: use el brújula para "declinar"

Modo de referencia ampliada de croquis
La brújula de referencia extendida también se puede cambiar en la vista en planta de un croquis [E]. Un toque largo en el botón brújula cambia la ventana al modo de "ampliar referencia". En este modo un el deslizamiento del dedo por la pantalla gira el brújula de referencia (el lienzo no se desplaza). Para salir de este modo levanta el dedo de la pantalla.

declinación de tomas
Cuando se descarga una nueva toma, TopoDroid establece su "declinación" automáticamente. Esto también se aplica a las tomas ingresadas manualmente, si la opción "declinar" en el cuadro de diálogo de toma se deja "sin configurar". De forma predeterminada, la "declinación" es IZQUIERDA, DERECHA o VERT (vertical), según el ángulo que haga el disparo con el brújula de referencia.
Si el ángulo es menor de 90° menos el umbral vertical declinado la toma se extiende a la DERECHA. Si es mayor de 90° más el declinar umbral vertical, la toma se extiende a la IZQUIERDA. De lo contrario, se extiende VERT.

El cuadro de diálogo declinar referencia permite cambiar el brújula de referencia, girándolo en sentido horario o antihorario. Alternativamente, el brújula de referencia se puede configurar usando los sensores de brújula de Android. Se muestra una advertencia si la precisión del sensor es baja o si hay una falla en el sensor.

También es posible fijar la "declinación" para las próximas tomas a IZQUIERDA o DERECHA, seleccionando los botones apropiados en el cuadro de diálogo.

declinar reproducción
La "declinación" de los disparos separados se determina de acuerdo con la de las tomas en la estación separada y el brújula entre la separación y las tomas. El splay se extiende como si se proyectara en el más cercano de los planos verticales que pasan por las tomas.
