NOMBRE DEL DISPOSITIVO

Este cuadro de diálogo se abre con un toque prolongado en una entrada de dispositivo en la ventana del dispositivo.

Muestra información sobre el dispositivo seleccionado y tiene un campo de texto para ingresar elseudónimo

. Si la configuración del segundo dispositivo está habilitada, hay una casilla de verificación para configurar el dispositivo como segundo dispositivo (solo DistoX).

Aceptar guarda el apodo en la base de datos y cierra el cuadro de diálogo.
Si la casilla de verificación está seleccionada, el dispositivo se establece como segundo dispositivo (solo DistoX).
