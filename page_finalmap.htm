EL MAPA FINAL

Esta sección describe el uso de Therion o cSurvey para redactar el mapa final.

Therión
Si desea compilar el proyecto Therion, debe proporcionar un archivo thconfig que puede ser tan simple como
fuente my_survey.th
exportar mapa -proy plan -o my_cave_p.pdf
exportar mapa -proj extendido -o my_cave_s.pdf

Debe haber exportado los datos topográficos como archivo Therion ".th" y los dibujos topográficos como archivos Therion ".th2".
El archivo de datos contiene comandos para ingresar archivos de croquis. Sin embargo, están comentados y es necesario descomentarlos.
El archivo de datos contiene comandos para definir un mapa para cada croquis (recorte). Para topografías simples, puede dejarlas comentadas.

Alternativamente, puede exportar los dibujos como archivos ".xvi" de Therion y volver a trazarlos usando xtherion.

cSurvey
Exporte el dibujo como archivo cSurvey ".csx".
Abra el archivo exportado con cSurvey: tendrá la hoja de datos llena de datos de la topografía y los dibujos correctamente alineados en el lienzo de dibujo decSurvey.
