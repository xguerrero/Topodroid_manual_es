PERMISOS de Android

Almacenamiento
TopoDroid guarda los datos topográficos, los dibujos y las calibraciones, en el almacenamiento externo.
No puede funcionar sin acceso de lectura/escritura al almacenamiento.

Internet
La conexión a la red solo es necesaria si desea instalar una traducción del manual del usuario.
TopoDroid no utiliza la red de ninguna otra manera y es seguro deshabilitar este permiso.

Bluetooth
TopoDroid requiere acceso bluetooth para comunicarse con los dispositivos.
Si los datos de la topografía se ingresan manualmente, se pueden usar sin bluetooth.
TopoDroid también usa el emparejamiento bluetooth, para emparejarse con dispositivos recién descubiertos. El descubrimiento y el emparejamiento se pueden hacer con la aplicación de "configuración".

Cámara
TopoDroid puede usar la cámara para tomar fotografías.
La administración de la cámara TopoDroid no es tan rica en funciones como la de una aplicación de "cámara", pero almacena la dirección de la vista (acimut e inclinación) en las imágenes.
Debería ser seguro usar TopoDroid sin este permiso.

Sonido
TopoDroid usa el micrófono para grabar notas de audio (para tomas y puntos de audio).
Si no se concede este permiso, la grabación de audio se desactiva.
Debería ser seguro usar TopoDroid sin este permiso.

Ubicación
TopoDroid utiliza el GPS del dispositivo para obtener la posición geográfica de un punto, pero puede importarlo desde Topógrafo móvil.
Debería ser seguro usar TopoDroid sin este permiso.
