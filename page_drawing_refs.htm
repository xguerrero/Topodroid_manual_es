REFERENCIAS DE DIBUJOS

Este cuadro de diálogo se abre desde el botón Referencia de la ventana de croquis.

La columna contiene las referencias del croquis,

  • tomas, línea media topografía (blanco)
  • pantallas (gris)
  • últimos despliegues [G]
  • nombres de las estaciones (violeta)
  • cuadrícula, métrica o yarda
  • barra de escala, barra de escala de referencia y flecha norte/hacia arriba
  • esquema, esbozo de otro dibujo

Las "últimas" visualizaciones se pueden resaltar en color azul claro [G]. Con el modo de descarga bajo demanda, estas son las presentaciones que se han descargado más recientemente. Con el modo continuo, estas son todas las visualizaciones descargadas durante el último intervalo de resaltado.

Hay tres rejillas,

  1. una cuadrícula con un espacio de 1 unidad (gris oscuro)
  2. una cuadrícula con un espacio de 10 unidades (gris)
  3. una cuadrícula con un espacio de 100 unidades (gris claro)

Las unidades de cuadrícula pueden ser 1m, 1y, 2ft, 0,1m. El valor predeterminado es 1 m.
TopoDroid muestra solo las cuadrículas que son apropiadas para el zoom de la visualización del croquis.

La barra de escala está adaptada a las unidades de cuadrícula (m, y, pies, dm) y tiene una pequeña flecha que indica la dirección Norte/Arriba. La leyenda en la barra de escala tiene el mismo tamaño que los nombres de las estaciones.

Si hay otros dibujos, se puede mostrar el contorno de otro dibujo (en gris) para ayudar a dibujar el dibujo Al marcar la casilla, se abre un nuevo cuadro de diálogo para elegir el dibujo que se mostrará o para borrar el contorno. pantalla.
El contorno de una sección X se puede mostrar en el punto de sección relativo. La visualización del contorno se activa y desactiva a través del cuadro de diálogo de edición de los puntos de sección.
La visualización de todos los contornos se borra cuando se cierra el croquis.

Desplazamiento y escala del croquis
Otra casilla de verificación [E/G] especifica si la referencia debe mantenerse fija cuando cambiando el dibujo con un toque de dos dedos. Esto se usa para ajustar un croquis importado de PocketTopo a la referencia en el caso de que se importe con una traducción. El dibujo se puede ampliar con un toque de tres dedos.

Vistas de dibujo (niveles)
Cuando las vistas de dibujo están habilitadas [G], hay otra columna, a la derecha, para activar y desactivar las vistas.
Un elemento de dibujo se muestra si pertenece al menos a una vista que está activada.
La vista Base (B) siempre está ACTIVADA. Las otras vistas son

  • Piso (F)
  • Rellenos (D)
  • Techo (C)
  • Artefactos (A)

Las vistas se indican con una letra mayúscula, B, F, D, C y A, como se indicó anteriormente.
