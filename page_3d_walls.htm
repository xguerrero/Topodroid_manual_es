VISOR 3D - Modelar paredes

Si el modelo tiene un número suficiente de separaciones, las paredes del modelo se pueden estimar. Hay tres algoritmos para calcular las paredes,
  • Recubrimiente
  • Recubrimiento convexo construye las paredes como una envolvente local casi convexa de los puntos finales separados
  • Powercrust funciona con una gran cantidad de separaciones e interpola las paredes a través de los extremos de las separaciones

Una vez calculada, la visualización de las paredes se activa y desactiva con el botón Paredes de la Ventana principalvisor 3D.

El cuadro de diálogo tiene un control deslizante para ajustar la transparencia de las paredes.
