EDICIÓN DE GEO-PUNTOS

Este cuadro de diálogo se abre tocando una entrada de punto fijo en la ventana Lista de geopuntos.

El cuadro de diálogo muestra

  • longitud, latitud y altitud geoide. Altitud del elipsoide WGS84 entre corchetes
  • punto nombre y comentario
  • declinación magnética en el punto (en la fecha actual)

Botones

  • Ver el punto en una aplicación de ubicación (siempre que tenga una instalada)
  • Borrar coordenadas convertidas. Esta acción es inmediata.
  • Convertir las coordenadas del punto geográfico a otro sistema de referencia (utilizando la aplicación Proj4)
  • Eliminar el registro de geopunto
Si las coordenadas del geopunto se han ingresado manualmente, se pueden editar. Las coordenadas tomadas con GPS o importadas de MobileTopographer no se pueden cambiar.

La altitud ortométrica se calcula a partir de la altitud elipsoidal utilizando el modelo de geoide EGM2008.

El botón Convertir calcula las coordenadas del punto geográfico en otro Sistema de Referencia de Coordenadas, utilizando la aplicación Proj4. La altitud del geoide se utiliza en la conversión. Las coordenadas convertidas se muestran y almacenan en la base de datos con el geopunto.

La declinación magnética se calcula utilizando el Modelo Magnético Mundial para 2020-2025 (esto se reemplaza con un WMM más reciente cuando se publique). La incertidumbre de la declinación del modelo WMM puede ser de hasta 0,4 grados en zonas templadas.
Marque la casilla guardar para establecer la declinación magnética en la encuesta.

El botón Guardar confirma los cambios en la base de datos.
