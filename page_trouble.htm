SOLUCIÓN DE PROBLEMAS DEL DISPOSITIVO

Emparejamiento de dispositivos
Antes de que TopoDroid pueda conectarse a un dispositivo, este debe estar emparejado con Android.
Puede emparejar Android con el DistoX activo a través del menú emparejar o puede usar la aplicación Configuración para emparejar el DistoX.

Si Android solicita el PIN de DistoX, ingrese 0000 (cuatro ceros).
Para el PIN de Sap5, introduzca 000000 (seis ceros).
Android no debería pedir el PIN de Bric4.

Una vez emparejado con el DistoX, el Android debería mantener el emparejamiento y no volver a solicitar el PIN. Si sigue solicitando el PIN cada vez que descarga datos del DistoX, intente usar la configuración de socket insegura.

Si tiene problemas para conectarse y descargar los datos con un dispositivo DistoX, intente habilitar la configuración Z6 workaround.
