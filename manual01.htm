Activity levels

TopoDroid tiene niveles de usuario, que se diferencian por el número de acciones que se habilitan.
  • At the basic En el nivel básico [B] puede dibujar las líneas, pero sin poderlas editar.
  • The normal El nivel normal [N] activa todas las funciones para un estudio detallado de la cueva.
  • The advanced el nivel [A] activa algunas funciones avanzadas.
  • The expert el nivel [E] activa todas las funciones.
  • The tester El nivel experimental [T] es sólo para desarrollo. Tiene las características experimentales habilitadas. Estas características están parcialmente depuradas y pueden bloquear fácilmente la aplicación. Por favor, informe de cualquier problema.

De manera predeterminada, TopoDroid solo tiene habilitadas las funciones normal , pero puede cambiar de nivel, según sus necesidades, sin salir del programa. Este manual describe el TopoDroid completo. Las características en un nivel superior a "básico" están marcadas con el nivel inicial, [N], [A], [E] o [T].
La configuración que se muestra en los cuadros de diálogo de configuración también depende del nivel de actividad.

Pantallas

TopoDroid tiene una pantalla principal desde donde se puede ir a otras funciones de la aplicación.
  • Survey La gestión de datos de la topografía (pantallas de topografía y lista de tomas).
  • Sketch El dibujo (pantalla de dibujo) es un sub-menu de la pantalla principal de la topografía.
  • DistoX Funciones (pantalla del dispositivo)
  • Calibración de DistoX (pantalla de datos de calibración y calibración) es un sub-menu de la pantalla principal del dispositivo.


Las pantallas de Topodroid tienen una barra de tareas en la parte superior para las acciones más frecuentes y un botón de menú en la parte superior derecha. Los cuadros de diálogo por lo general tienen un botón de confirmación (OK o Guardar). Si desea cerrar el diálogo, o cancelar alguna acción, toque el botón ATRÁS de Android. Si todo es correcto pulse ok o el botón Guardar. Ok guarda los datos de entrada y cierra el cuadro de diálogo, mientras que el segundo botón, guarda los datos de entrada y mantiene el cuadro de diálogo abierto para una nueva entrada. Los diálogos de información no tienen ningún botón.

En cada pantalla TopoDroid sigue teniendo el botón de MENÚ que abre el cuadro de diálogo para seleccionar diferentes opciones. el botón abre Manual de usuario y páginas.

Tamaño de texto y de los botones.

El ajuste "Tamaño de texto" especifica el tamaño del texto de la lista, en particular, la lista de las topografías, la de las tomas y la de los datos de calibración
Hay dos opciones para el tamaño de los botones: "pequeño" y "grande". El primero es adecuado para teléfonos móviles (pantalla de 5 "), y es el valor predeterminado. El segundo es adecuado para tabletas (pantalla 7 ').

Drawing tools (palette)

TopoDroid comes with a considerable number of drawing tools bundled in. By default only the most basic tools are enabled. You can enable more tools as the need arises, with the "palette" button on the main window. By this means you can get used to the TopoDroid "iconography" a bit at a time.
