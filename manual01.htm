Activity levels

TopoDroid tiene niveles de usuario, que se diferencian por el número de acciones que se habilitan.
  • At the basic En el nivel básico [B] puede dibujar las líneas, pero sin poderlas editar.
  • The normal El nivel normal [N] activa todas las funciones para un estudio detallado de la cueva.
  • The advanced el nivel [A] activa algunas funciones avanzadas.
  • The expert el nivel [E] activa todas las funciones.
  • The tester El nivel experimental [T] es sólo para desarrollo. Tiene las características experimentales habilitadas. Estas características están parcialmente depuradas y pueden bloquear fácilmente la aplicación. Por favor, informe de cualquier problema.

De manera predeterminada, TopoDroid solo tiene habilitadas las funciones normal , pero puede cambiar de nivel, según sus necesidades, sin salir del programa. Este manual describe el TopoDroid completo. Las características en un nivel superior a "básico" están marcadas con el nivel inicial, [N], [A], [E] o [T].
La configuración que se muestra en los cuadros de diálogo de configuración también depende del nivel de actividad.

Windows

TopoDroid has a main window from which you can move to other app activities. The main activities are:
  • Survey data management (survey and shot-list windows).
  • Sketch drafting (drawing window) is a sub-activity of the Survey activity.
  • DistoX functions (device window)
  • Calibration of DistoX (calibration and calibration-data windows) is a sub-activity of the Device activity.


Topodroid windows have a scrollable button bar at the top, for the most frequent actions, and a menu button at the top-right. The dialogs usually have a confirmation button (OK or Save). They can have also a "back" (or "cancel") button. If not present, tap the Android BACK button to close the dialog. If there are both the OK and the Save button, the former saves the input data and closes the dialog, while the latter saves the input data and keep the dialog open for a new input.

In every TopoDroid window the hardware MENU key opens the User Manual pages.

Text and Buttons size

The "Text Size" setting specifies the dimension of the list text entries, in particular the list of surveys, that of shots, and that of calibration data.
There are five choices for the size of buttons: "small", "normal", "medium", "large", and "huge". The "normal" size is suitable for cell phones (5' screen), and is the default. The "large" size is for tablets (7' screen).

Drawing tools (palette)

TopoDroid comes with a considerable number of drawing tools bundled in. By default only the most basic tools are enabled. You can enable more tools as the need arises, with the "palette" button on the main window. By this means you can get used to the TopoDroid "iconography" a bit at a time.
