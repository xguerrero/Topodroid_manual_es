[4] PANTALLA DE CALIBRACIÓN

Tocar el menú de calibración en la pantalla del dispositivo muestra las listas de las calibraciones del DistoX activo. Puede abrir una calibración o crear una nueva. En ambos casos se accede desde la ventana de calibración.

La ventana de calibración tiene campos de texto para:

  • El nombre. Cada calibración debe tener un nombre único.
  • La fecha.
  • Una descripción.
Los botones de radio para el algoritmo de calibración. De forma predeterminada, esta opción está configurada en "Auto" (es decir, deje a TopoDroid decidir qué algoritmo utilizar).

Botones

  • Guardar la información de calibración a la base de datos.
  • Abrir la ventana Datos de calibración.
  • Mostrar los coeficientes de calibración. Esto muestra los coeficientes calculados previamente y se almacenan en la base de datos.

Menús

  • Exportación de calibración
  • Eliminar la calibración y sus datos de la base de datos (Acción no recuperable).
  • Ajustes
  • Ayuda

ajustes de calibración

  • Política de grupo: cómo dividir los datos de calibración en grupos [predeterminado "Grupo 4"]
  • Precisión grupo: la máxima distancia angular entre datos consecutivos para que estén en el mismo grupo, cuando la "distancia" se utiliza la política de grupo [predeterminado 40 min 0]
  • Error: el cálculo de los coeficientes se termina cuando el coeficiente de variación de cada una de las iteraciones durante un paso es menor que este valor predeterminado [10-6 i, min 0].
  • Iteraciones: el número máximo de iteraciones del algoritmo de calibración [predeterminado 200, 50 min].
  • Datos sin procesar: por defecto TopoDroid muestra el rumbo, inclinación y balanceo. Seleccione esta opción si desea ver también los datos de calibración raw, decimales o hexadecimales. [Default "no"]
  • Algoritmo: puede ser 0 (auto), 1 (lineal), 2 (no lineal). Si un algoritmo de calibración queda "auto" y esta configuración no es "auto", el algoritmo de la configuración se utiliza para calcular la calibración. Si ambos son "auto", el algoritmo es elegido automáticamente por DistoX de TopoDroid, dependiendo del modelo y el firmware. "Auto" [default]
