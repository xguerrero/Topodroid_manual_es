ADMINISTRADOR DE PROYECTOS - Ventana de proyecto

La cueva Ventana de proyecto se abre tocando un nombre de proyecto en la lista de la Ventana de administrador.

Muestra la lista de topografías que componen el proyecto. La lista está vacía si el proyecto no tiene topografías.
Cada topografía se puede seleccionar para otras acciones, tocando su casilla de verificación.

Menús

  • Cerrar la ventana
  • Exportar el proyecto de la cueva (Therion o Survex)
  • Eliminar el proyecto de la cueva
  • Ayuda

Botones
  • insertar topografías
  • eliminar las topografías seleccionadas
  • 2D visualización de las topografías seleccionadas, en vista en planta
  • enumere el proyecto equivale
  • Vista 3d del proyecto

El botón insertar muestra la lista de topografías en la base de datos de TopoDroid. Las topografías se agregan al proyecto tocando sus casillas de verificación y confirmando con el botón Aceptar.

Vista 3D del proyecto
El proyecto se muestra en el visor 3D a partir de la primera topografía.
Los siguientes levantamientos se muestran en 3D solo si están (directa o indirectamente) conectados a la primera topografía.

Si se georreferencian varios levantamientos, se pueden mostrar todos en 3D, junto con el topografías conectadas.

Vista 2D de topografías
El levantamiento seleccionado se muestra en una vista en planta, con los nombres de la línea media y la estación. Cada topografía tiene un color diferente. Después de un tiempo, los colores se repiten, por lo tanto, debe mostrar solo dos o tres topografías a la vez.
Las equivalencias se muestran como líneas discontinuas rojas y las estaciones equiparadas tienen un fondo redondo.

La vista se puede desplazar y ampliar.
Se puede agarrar una topografía (pulsar una estación: se rodea con un círculo) y arrastrarla mientras las otras topografías no se mueven. De esta forma podrá superponer estaciones de diferentes levantamientos que desee equiparar. Cuando se superpongan, encierre en un círculo ambos tocando el punto: luego puede insertar la equivalencia entre ellos con el menú "agregar equivalencia".

Botones

  • Agregar equivalencia: abre un cuadro de diálogo para agregar manualmente una equivalencia entre levantamientos del proyecto de la cueva.
  • Lista de equivalencias entre topografías del proyecto de la cueva
    • Los nombres de los grupos están en inglés.
