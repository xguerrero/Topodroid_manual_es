GPS GEO-PUNTO

Este cuadro de diálogo se abre desde el botón GPS de la ventana Geo-points.

Para usar el GPS, el servicio de "Ubicación" de Android debe estar habilitado. Por razones de seguridad, una aplicación genérica, como TopoDroid, no puede habilitar el GPS. Si no está habilitado, TopoDroid muestra un cuadro de diálogo que le pregunta si desea habilitarlo (a través de la aplicación "Configuración"). Recuerde desactivarlo cuando haya terminado, para ahorrar batería.

Campos:

  • nombre de la estación
  • comentario
Botones:
  • GPS inicia/detiene la ubicación del punto geográfico
  • Ver el punto en una aplicación de mapa
  • Guardar el punto
Mientras busca satélites, el botón "GPS" se vuelve rojo (o verde) y muestra el número de satélites a la vista. Cuando hay suficientes satélites a la vista, el botón se vuelve verde y las coordenadas se muestran debajo. Debe esperar hasta que los valores de las coordenadas sean estables y el error de ubicación sea lo suficientemente pequeño. Luego detenga la ubicación del GPS tocando el botón nuevamente.
