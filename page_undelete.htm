RECUPERACIÓN DE OBJETOS y BÚFER DE TOMAS

Este cuadro de diálogo se abre desde el menú Recuperar de la ventana de datos de la topografía.

Recuperación de artículos
El cuadro de diálogo se puede utilizar para recuperar los elementos, tomas o dibujos eliminados de la topografía, así como datos de "sobreimpulso" y datos de "comprobación de calibración".

El botón "Estado" selecciona qué elementos se muestran. Tocándolo, la pantalla alterna entre tomas "eliminadas", "sobrepasadas", tomas de "comprobación de calibración" y dibujos "eliminados".
Seleccione los elementos para recuperar tocando sus casillas de verificación. Para finalizar y recuperar los elementos seleccionados, toque el botón "Aceptar".
Solo se recuperan los elementos seleccionados del tipo expuesto.

Una recuperación de croquis restaura tanto el plano como la vista de perfil al mismo tiempo.

Búfer de disparos [T]
Si el búfer de tomas no está vacío, los datos pueden eliminarse del búfer y adjuntarse a la topografía
Las tomas se agregan en el orden en que se agregan al búfer, o se pueden ordenar por el índice [predeterminado].
