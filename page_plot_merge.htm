COMBINACIÓN Y DIVISIÓN DE CROQUIS

En el cuadro de diálogo Cambiar nombre/eliminar croquis puede cambiar el nombre o eliminar el croquis abierto.

Si dividir/combinar [G] está habilitado, también hay botones para dividir el croquis (es decir, extraer parte de los dibujos creando un nuevo croquis) y combine los dibujos de otro croquis en el croquis actual.

División de croquis
También hay un botón para dividir el croquis. Esto crea un nuevo croquis poblado con elementos seleccionados del croquis actual. Para seleccionar estos elementos, trace un borde que los rodee en la ventana de croquis.
Puede especificar el nombre del nuevo croquis. Sin embargo, su origen es el mismo que el del croquis actual. Asimismo, tiene el mismo tipo de vista de perfil que el croquis actual.
Si la ventana de croquis muestra la vista en planta, la nueva vista en planta del croquis se rellena con los elementos seleccionados y la vista de perfil esta vacio. Lo contrario, si la ventana de croquis muestra la vista de perfil.
Solo los elementos visibles se seleccionan para la división.

Puede elegir si desea mover los elementos seleccionados al nuevo croquis o copiarlos en él.

Después de esta operación, la ventana de croquis muestra el nuevo croquis.

Fusión de croquis
Si se muestra el contorno de otro croquis, puede fusionarlo con el croquis actual.
Los elementos del croquis perfilado se copian en el croquis actual.
