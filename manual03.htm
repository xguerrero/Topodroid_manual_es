[3] Pantalla del dispositivo

Antes de realizar el levantamiento, debe conectar y emparejar el DistoX con Android y calibrarlo. Toque el botón DistoX desde la pantalla principal para abrir la pantalla del dispositivo. La lista de dispositivos DistoX conocidos se muestra en el medio. TopoDroid funciona con un DistoX a la vez. Para seleccionarlo, toca su entrada en la lista. El nombre del DistoX seleccionado se muestra en la parte superior, justo debajo de los botones. Si no se selecciona DistoX, aparece un mensaje rojo que dice "Dispositivo no seleccionado".

Si la lista está vacía, su DistoX debe estar conectado y sincronizado con Android. Esto puede hacerse con TopoDroid (menús Scan y emparejar) o con los ajustes de la aplicación.

Apodos DistoX
Por defecto TopoDroid utiliza la dirección de BT como nombre para un DistoX. DistoX puede renombrarse con apodos para distinguir entre varios dispositivos más fácilmente.

Nota. Los botones y los menús marcados con asterisco [*] necesitan el DistoX encendido.

Ajustes del dispositivo

  • Bluetooth: Activa, desactiva o pregunta al inicio [por defecto].
  • Modo de Conexión: permanece conectado al DistoX para descargar datos tan pronto como sean tomados, o descargar las tomas agrupadas, "Demanda". El modo "multi" es como "demanda", pero pulsando largo el botón permite cambiar a DistoX. El modo de conexión sólo se aplica a los datos: otras funciones de DistoX utilizan un modo de consulta de un solo uso [por defecto "demanda"]
  • Autoconectar : vuelve a conectar automáticamente si la conexión se pierde en modo "continuo" de la descarga de datos [por defecto "no"]
  • Emparejar tipo: normal o "inseguro" [predeterminado "normal"].
  • Z6-solución arregla un error de tiempo de espera de descarga en un teléfono Z6, y posiblemente en otros dispositivos. Es seguro dejar activado "ON" [por defecto]
  • Retardo en la conexión : Retardo [1/10 seg] antes de intentar conectarse al DistoX [por defecto] 0, sin ningún retardo.
  • Emparejamineto automatico : TopoDroid debería intentar emparejar con el DistoX seleccionados después de escanear [por defecto "no"]
  • Pausa de datos : pausa después de una descarga de datos [predeterminado 100 ms]
  • Tiempo de espera de datos : el tiempo de espera para que los datos esten listos [por defecto] 500 ms
  • Tiempo de espera de láser : el tiempo de espera después de girar el láser en 1000 ms [por defecto]
  • Tiempo de espera de disparo : el tiempo de espera después de tomar una medida [por defecto] 4000 ms
