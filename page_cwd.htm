Carpetas de proyectos de TopoDroid

Para gestionar proyectos de levantamiento de cuevas que impliquen varias sesiones de levantamiento, se recomienda utilizar un directorio de trabajo específico (carpeta) para cada proyecto de cueva.
El nombre de cualquier directorio de trabajo debe comenzar con "TopoDroid" (independientemente de las mayúsculas y minúsculas).
Las carpetas del proyecto se crean dentro de la carpeta base de TopoDroid "Documentos/TDX".

Los archivos de símbolos de dibujo y los archivos relacionados con los dispositivos (base de datos, firmware, volcados de memoria, calibraciones) se almacenan solo en la carpeta privada de "archivos" de la aplicación (en el almacenamiento externo). Las carpetas del proyecto contienen solo la base de datos de topografías del proyecto y las carpetas de encuestas.

La elección de la carpeta del proyecto actual es una de las configuraciones generales de la aplicación en la ventana principal.
