INSERCIÓN DE GEO_POINT

Este cuadro de diálogo se abre desde el botón Agregar en la ventana iGeo-points.

Este cuadro de diálogo tiene cuatro campos de texto:

  • longitud
  • latitud
  • altitud elipsoidal
  • altitud ortométrica
A la izquierda de la longitud hay un botón para especificar si la longitud es este (positiva) o al oeste (negativo). Al tocar el botón, se alterna entre "E" y "W".

A la izquierda de la latitud hay un botón para especificar si la latitud es norte (positiva) o sur (negativo). Al tocar el botón, se alterna entre "N" y "S".

La longitud y la latitud se pueden ingresar como grados-minutos-segundos (dd:mm:ss.ss) o como grados decimales. Separadores alternativos para dos puntos son espacio, comillas simples y el símbolo de grado. Los separadores alternativos para el punto decimal son la coma y la barra oblicua.

La longitud y la latitud pueden tener un signo negativo.

Las altitudes se ingresan en metros. Cuando se especifican la longitud y la latitud, ingresando una de las dos altitudes hace que TopoDroid calcule y muestre automáticamente el otro.

Botones:

  • Portapapeles: las coordenadas guardadas en el portapapeles como "lat,long[,alt]" se copian del portapapeles al respectivos campos de texto. Con este botón puede usar la aplicación GPS-Impetus para obtener las coordenadas (la altitud debe ser introducido manualmente).
  • Ver invoca una aplicación de mapas (siempre que tenga una instalada) para mostrar el punto.
  • con Proj4 puede ingresar las coordenadas en otro Sistema de Referencia de Coordenadas. La aplicación Proj4 se invoca, y debe seleccionar el CRS e ingresar las coordenadas en el lado izquierdo: longitud/Este, latitud/Norte y altitud del geoide. Las unidades de altitud y coordenadas cartográficas son metros. Las unidades de coordenadas geográficas pueden ser grados decimales o grados-minutos-segundos. A continuación, toca la flecha de conversión y las coordenadas se convierten a WGS84 geográficas y se envían de vuelta a TopoDroid.
