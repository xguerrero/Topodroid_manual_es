MEDIDAS DE SENSORES

Este cuadro de diálogo se abre desde el botón Sensor en el cuadro de diálogo secundario Editar toma.

Tiene una lista de casillas de verificación para seleccionar un sensor incorporado o un sensor externo (en este caso debe ingresar el nombre del "sensor").

Hay tres campos de texto.

  • nombre del sensor. Cualquiera de los sensores incorporados enumerados en la lista o debe escribirse si selecciona un sensor "externo"
  • valores Actualizados automáticamente para sensores integrados. Introducido manualmente para sensores "externos".
  • comentario
Guardar guarda la medida en la base de datos.

La tecla ATRÁS de Android cierra el cuadro de diálogo sin guardar.
