SENSOR MEASUREMENTS

This dialog is opened from the Sensor button in the secondary Shot edit dialog.

It has list a checkboxes to select a builtin sensor, or an external sensor (in this case you must enter the "sensor" name).

There are three text fields

  • name of sensor. Either one of the listed builtin sensors, or it must be typed in if if you select an "external" sensor
  • values Automatically updated for builtin sensors. Manually entered for "external" sensors.
  • comment
Save stores the measurement in the database.

The Android BACK key closes the dialog without saving.
