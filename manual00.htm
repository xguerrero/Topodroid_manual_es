TopoDroid v.3.4 Manual de usuario

2017-07-01 - marco.corvi@gmail.com - Traducción al español Manu Guerrero manuguerreros@gmail.com

TopoDroid es una aplicación Android para topografiar cuevas. TopoDroid toma datos sin procesar y te ayuda a organizarlos. También realiza un dibujo de la topografía insitu. Puedes realizar varios dibujos mientras estás topografíando. Esto reduce la probabilidad de error al realizar el dibujo, y ayuda a detectar errores en los datos mientras se encuentra todavía en la cueva. TopoDroid tiene muchas funciones para ayudarle a obtener el máximo rendimiento en el trabajo que realizamos en la cueva. De esta manera el trabajo topográfico está casi terminado antes de salir de ella.

TopoDroid hace que el tiempo que pasa en la cueva topografiando sea más eficaz. Está diseñado para trabajar con una sola topografía. No administra proyectos de topografías complejas que incluyen mas de una topografía. El propósito de TopoDroid es ayudar al topógrafo a recoger los datos con más precisión, así como para hacer la tarea de topografía más agradable, no para realizar la topografía final de la cueva. Para ello ya existen buenos programas de PC.

TopoDroid está específicamente diseñado para trabajar con el DistoX y DistoX2, aunque los datos de la topografía se pueden introducir manualmente.

TopoDroid utiliza una orientación vertical fija de la pantalla, no se puede rotar.

Características de hardware

TopoDroid utiliza las siguientes características de hardware: "bluetooth", "cámara", "micrófono", "GPS", "multitouch". Las funciones de TopoDroid que requieren una característica que no está presente en su dispositivo Android son desconectadas automaticamente.

Errores y bloqueos

Debido a los recursos limitados, la aplicación no se puede probar plenamente. Es aconsejable instalar actualizaciones de TopoDroid, con un gestor de aplicaciones que mantiene la aplicación actualizada. Por este medio, puede volver con seguridad a una versión anterior que se sienta cómodo, en caso de encontrar problemas con la última versión. No se olvide de informar de sus problemas para que puedan ser tenidos en cuenta en la próxima versión. Los informes de errores se analizan con prontitud y la aplicación se actualiza con frecuencia.
  • Si usted recibe un error o cierre inesperado de la aplicación, por favor presente un informe de errores a través de su sistema Android.
  • Si usted encuentra un error o la aplicación se comporta erróneamente, envíeme un correo electrónico.
  • Si usted tiene una pregunta o una solicitud de alguna función, envíe un email a topodroid.

Errores de Traducción
