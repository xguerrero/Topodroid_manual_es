DATOS DE CALIBRACIÓN del DistoX

Este cuadro de diálogo se abre tocando una entrada de datos en la ventana de datos de calibración.

  • Rumbo, inclinación, balanceo
  • calibración error
  • grupo campo de entrada de número
  • Guardar los cambios

Los botones son

  • reasignar los grupos a partir de estos datos
  • marcar los datos como "eliminados"

El grupo de elementos se puede establecer o cambiar.

  • positivo: los datos participan en el cálculo de los coeficientes de calibración
  • cero: los datos no entran en el cómputo
Los datos con el mismo número de grupo positivo forman un grupo.

Los datos eliminados no se eliminan de la base de datos.
Normalmente no se muestran en la lista de datos de calibración, no se tienen en cuenta en la asignación automática de grupos y no entrar al cómputo de los coeficientes de calibración.
