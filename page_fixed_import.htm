IMPORTACIÓN DE GEO_PUNTOS

Este cuadro de diálogo se abre desde el botón Importar de la ventana Geo-points.

Con este diálogo puede importar en TopoDroid las coordenadas de un punto de un archivo de lista de puntos de MobileTopographer.
Si no selecciona un archivo de lista de puntos de MobileTopographer, el comportamiento de la aplicación no está definido.

  1. Abra MobileTopographer y encuentre las coordenadas del punto.
  2. Cuando haya alcanzado una precisión suficiente en las coordenadas, guarde el punto.
  3. También puede ingresar las coordenadas del punto a mano (y guardar el punto).
  4. Guarda la lista de puntos. La lista se guarda en "MobileTopographer/pointlist"
  5. Importa las coordenadas en TopoDroid. Los puntos guardados se muestran en una lista; toque el punto para importar.
Opcionalmente, puede ir a la lista de puntos guardados en MobileTopographer, y mueve el punto que acabas de llegar a la parte superior con las flechas.
