Subdirectorios TopoDroid

Aquí hay algunos antecedentes sobre cómo se almacenan los datos topográficos, los datos de calibración y los dibujos.

La aplicación crea su directorio base predeterminado en la memoria externa pública principal, "Documentos/TDX". Este directorio no se elimina cuando desinstala la aplicación. En este directorio la aplicación crea la carpeta de topografías "TopoDroid" con la base de datos SQLite "distox14.sqlite" que almacena los datos de la topografía, los archivos de croquis de topografías y otros archivos de topografías.

La aplicación también crea subdirectorios para los diferentes tipos de archivos.
Los siguientes subdirectorios se crean solo en el directorio privado de la aplicación, es decir "Android/data/com.topodroid.TDX/files" en la memoria externa principal.

  • bin: archivos de firmware (solo DistoX2)
  • ccsv: archivos de texto CSV de calibraciones exportadas (solo DistoX)
  • dump: archivos de volcado de memoria (solo DistoX)
  • símbolo: archivos de herramientas de dibujo (puntos, líneas, áreas)
  • man: traducción del manual del usuario
"bin" se usa para almacenar archivos de firmware, tanto los archivos de fábrica incluidos en TopoDroid como los volcados de firmware leídos desde un DistoX2.
La base de datos "device10.sqlite", con los datos de calibración y la información de los dispositivos se encuentra en este directorio.

Para cada topografía se crea una carpeta de topografías (nombrada según el nombre de la topografía), con las siguientes carpetas:

  • audio: audio comentarios (archivos wav)
  • nota: archivo de texto de anotaciones de la topografía
  • foto: imágenes de la topografía (archivos jpg)
  • tdr: archivos de dibujo binarios
  • fuera: archivos exportados
