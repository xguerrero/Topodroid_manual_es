[2] VENTANA PRINCIPAL

La ventana principal tiene cuatro botones en la parte superior, y la lista de las topografías debajo. Esta lista está vacía hasta que cree una topografía nueva.

La primera vez que inicia la aplicación, se muestra un mensaje de bienvenida. Este se puede volver a ver en cualquier momento desde el menú principal, en el botón Acerca de

Para salir de la pantalla principal de TopoDroid debe hacer doble click en el botón BACK de su dispositivo.

COMPORTAMIENTO

  • Pulsación corta sobre el nombre de una topografía: pantalla de datos de la topografía.
  • Pulsación larga sobre el nombre de una topografía: pantalla de información topografía.

BOTONES

  • Dispositivo: cambia a la pantalla de dispositivos
  • Nueva topografíaCrea una nueva topografía
  • Importar Importar una topografía o un archivo ZIP
  • Palette Paleta para cargar los iconos por defecto en el dibujo.
  • Manager Abre la app ThManager.

MENúS

  • Palette [A] abre un cuadro de diálogo para cargar conjuntos de herramientas de dibujo adicionales
  • Cosurvey [E] abre el cuadro de diálogo para conectarse con la aplicación TopoDroid en otro Android para realizar la union de otra topografía
  • Settings: all the program settings
  • Logs settings [E]. Logging is mostly for debugging app error
  • About: the program welcome message
  • Help shows a brief descriptions of buttons and menus. At the top-right corner there is a button that opens this user manual. Each TopoDroid window has a similar Help menu

SETTINGS

  • Current work directory [default "TopoDroid"]
  • Language
  • Text size of the lists items [default 14, min 1]
  • Buttons size: small, normal (for 5" screen), medium, large (for 7" tablets), huge. [default small]
  • Keyboard enables or disables the TopoDroid keypads.
  • Activity level: basic, normal, advanced, expert. [default normal].
  • Cosurveying [E] activate/deactivate the cosurveying service (very experimental).

From the "main window" settings dialog you can go to the other settings dialogs

  • Import/Export
  • Survey data
  • Sketching
  • Devices

If bluetooth is not on, the app asks whether to turn it on. Without bluetooth it cannot communicate with the DistoX and the survey data can only be entered manually. In this case the Download button in the Shot window is hidden, and that in the Sketch window is replaced by the Add button.
