MEMORIA DistoX

Este cuadro de diálogo se abre desde el botón Memoria de la ventana del dispositivo.

Volcado de memoria

El botón Dump lee el contenido de la memoria DistoX. Leer la memoria del DistoX directamente es un proceso lento, y el DistoX debe mantenerse "ENCENDIDO" mientras lo hace. Es recomendable leer una pequeña porción de la memoria, digamos 20 entradas, a la vez: ingrese el índice de memoria de datos inicial y final en los dos campos de texto a la derecha. Los datos se leen desde el índice inicial hasta uno antes del índice final. Para DistoX2, los índices de memoria van desde 0 incluido hasta 1064 excluido.
Para la versión 1 de DistoX, la memoria varía de 0 incluida a 4096 excluida.

Si ha introducido un nombre de archivo, el volcado de memoria también se guarda en un archivo (en la carpeta "dump").

El resultado del volcado de memoria se muestra en una lista en la parte inferior del cuadro de diálogo. Cada línea contiene

  • la ubicación de la memoria
  • un código de letras: tiro de frente ('d'), punto de referencia ('b'), datos de calibración ('g' o 'm')
  • los valores de la memoria.

El código de letras mayúsculas indica datos que aún no se han descargado.
Las tomas se muestran como longitud (metros), rumbo y clino (grados).
Los datos de "precisión" del DistoX2 no se muestran.
Para los datos de calibración, se muestran los valores brutos. Los datos 'm' no se muestran para el DistoX2.
Las entradas sin un tipo reconocido se marcan con un código de signo de interrogación ('?').
