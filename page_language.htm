IDIOMAS

De manera predeterminada, TopoDroid usa el idioma predeterminado del sistema si TopoDroid contiene la traducción adecuada. De lo contrario, se utiliza el inglés. Puede anular esta opción seleccionando otro idioma entre los admitidos:
  • Inglés
  • Chino (H.J. Luo)
  • Húngaro (B. Holl)
  • Italiano (F. Toso)
  • Ruso (A. Kozhenkov)
  • Español (M. Guerrero)

La elección del idioma es una de las configuraciones generales de la aplicación.

Advertencia Debido al almacenamiento en caché, es posible que Android no use el idioma seleccionado para todos los textos. Una solución alternativa es un reinicio en frío (una parada forzada seguida de una apertura).

Advertencia
Configurar el idioma con la aplicación "Configuración" cambia el idioma de TopoDroid, sin embargo, los elementos ya cargados utilizando la configuración de idioma de TopoDroid (listas de menús, cuadros de diálogo, ...) NO se recargan con el idioma del sistema.
