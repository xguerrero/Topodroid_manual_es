EDITAR LA MEDICIÓN DEL SENSOR

Este cuadro de diálogo se abre tocando una entrada de medición del sensor en el cuadro de diálogo Lista de sensores.

Muestra información sobre la medición (título, tipo, disparo y valor) y un campo para cambiar el sensor comentario.

Botones

  • Aceptar guarda los cambios y cierra el cuadro de diálogo
  • Eliminar elimina la medición

Eliminar no elimina la medición del sensor de la base de datos, sino que la marca como "eliminada". Se puede recuperar con una aplicación SQLite.
