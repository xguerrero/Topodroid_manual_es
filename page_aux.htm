Aplicaciones auxiliares

Cuando trabaje con TopoDroid, puede encontrar útiles algunas aplicaciones complementarias y auxiliares:
  • Proj4 es una aplicación de conversión de coordenadas (muy simple), utilizada por TopoDroid para convertir valores de latitud larga a otros sistemas de coordenadas
  • MobileTopographer es una aplicación de geolocalización. Puede usarlo para tomar las coordenadas del punto y hacer que TopoDroid las lea desde el archivo de puntos en la lista de puntos de MobileTopographer.
  • CrowdMag es una aplicación del Centro Nacional de Datos Geofísicos de la NOAA, útil para obtener información sobre el estado actual del campo magnético terrestre.
  • una aplicación para tomar fotos (generalmente hay una aplicación Cámara)
  • un administrador de archivos decente, como MiXplorer, para ver los archivos y hacer copias de seguridad
  • SQLiteViewer (o cualquier otra aplicación de administración de SQLite) puede ser útil para inspeccionar las bases de datos de TopoDroid

Además, dependiendo de cómo procese sus topografías, es posible que desee instalar

  • un visor/editor de imágenes (generalmente hay una aplicación preinstalada)
  • un visualizador/editor de DXF
  • un visor/editor de hojas de cálculo
  • un visor SVG)
  • un archivador ZIP
  • GoogleEarth
  • un visor de archivos de seguimiento

Proj4
TopoDroid almacena coordenadas de puntos en el sistema de referencia geográfica de latitud larga WGS-84. Las altitudes se almacenan tanto referidas al elipsoide como al geoide.

TopoDroid utiliza la aplicación Proj4 de dos maneras para transformar coordenadas entre WGS-84 y otros sistemas de coordenadas.

  • para convertir las coordenadas de un punto de latitud larga (WGS-84) a otro sistema de coordenadas de referencia.
  • para ingresar las coordenadas de un punto en un sistema de coordenadas diferente.
