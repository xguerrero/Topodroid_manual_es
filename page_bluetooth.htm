BLUETOOTH

El botón Bluetooth está presente en la ventana de datos de tomas, en la ventana de dibujo y en la ventana de datos de calibración.

Para DistoX y Sap5, el botón Bluetooth restablece la conexión Bluetooth.

Para el DistoX2 abre un menú desplegable [T]:

  • Restablecer la conexión Bluetooth
  • encender/apagar el láser
  • tomar una foto o datos de calibración
  • hacer tres tomas seguidas, es decir, una visual (ventana data o dibujo)
En las ventanas dato y dibujo, las tomas se descargan inmediatamente si el modo de comunicación del dispositivo es "continuo".
En la ventana de datos de calibración, los datos se descargan inmediatamente dependiendo de la configuración.

Después de encender el láser, hay una breve pausa antes de disparar. Del mismo modo, hay una pausa entre tomas cuando se toman tres. La duración de estas pausas se define en la configuración.

Para Bric4, el botón Bluetooth abre un menú desplegable [T]:

  • Restablecer la conexión Bluetooth
  • encender/apagar el láser
  • tomar lecturas
  • inicia un modo de escaneo (solo si la conexión Bric4 está configurada para recibir los datos completos)
  • apagar el dispositivo
  • restablecer el contador de disparos del Bric4 (solo si la topografía no tiene datos)
